   <h2></h2>
   <div class="legend"><span style="">
         <p>Требуется найти в бинарном векторе самую длинную последовательность единиц и вывести её длину. </p></span><p>Желательно получить решение, работающее за линейное время и при этом проходящее по входному массиву только один раз.</p>
   </div>
   <h2>Формат ввода</h2>
   <div class="input-specification"><span style="">
         <p>Первая строка входного файла содержит одно число <span class="tex-math-text">n</span>, <span class="tex-math-text">n ≤ 10000</span>. Каждая из следующих <span class="tex-math-text">n</span> строк содержит ровно одно число — очередной элемент массива.
         </p></span></div>
   <h2>Формат вывода</h2>
   <div class="output-specification"><span style="">
         <p>Выходной файл должен содержать единственное число — длину самой длинной последовательности единиц во входном массиве.</p></span></div>
